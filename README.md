# Vehicle Maintenance Manager — Legal Pages

This repository hosts the Privacy Policy and Terms & Conditions for the Vehicle Maintenance Manager Android app.

## Files
- `index.html` — Privacy Policy (rename privacy_policy.html to this)
- `terms.html` — Terms & Conditions
- `play_store_description.txt` — Play Store listing content

## Live URLs after setup
- Privacy Policy: `https://YOUR_USERNAME.github.io/vehicle-maintenance-policy/`
- Terms & Conditions: `https://YOUR_USERNAME.github.io/vehicle-maintenance-policy/terms.html`

## Setup Steps (5 minutes)

### Step 1 — Create GitHub Repository
1. Go to github.com → New repository
2. Name: `vehicle-maintenance-policy`
3. Set to **Public**
4. Click **Create repository**

### Step 2 — Upload Files
1. Click **uploading an existing file**
2. Upload these files:
   - `privacy_policy.html` → rename to `index.html`
   - `terms.html`
3. Click **Commit changes**

### Step 3 — Enable GitHub Pages
1. Go to **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** → **/ (root)**
4. Click **Save**
5. Wait 2 minutes → your pages are live!

### Step 4 — Update Email Address
In both `index.html` and `terms.html`, replace:
`your@email.com` with your real email address
